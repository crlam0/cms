<p align=left>
<font class=small>
<b>Тэги, доступные для использования в шаблонах:</b><br>
<b>[%PHP_SELF%]</b> - Имя текущего скрипта<br>
<b>[%PHP_SELF_DIR%]</b> - Каталог текущего скрипта<br>
<b>[%BASE_HREF%]</b> - URL сайта<br>
<b>[%SUBDIR%]/<b> - Подкаталог сайта<br>
<b>[%title%]</b> - Заменяется на $tags[title]<br>
<b>[%func(title,param)%]</b> - Запускает функцию title с параметром param<br>
<b>[%var(title)%]</b> - Заменяется на значение переменной $title<br>
<b>[%include(title)%]</b> - Подключает PHP файл title<br>
<b>[%file(title)%]</b> - Вставляет файл title<br>
<b>[%template(title)%]</b> - Вставляет шаблон title<br>
<b>[%settings(title)%]</b> - Вставляет значение title из таблицы настроек<br>
<b>[%inner_content%]</b> - Вставляет контент, передаваемый шаблону<br>
<b>[%block(title)%]</b> - Вставлет блок title<br>
<br>
<b>Тэги, Используемый при работе с таблицами:</b><br>
<b>[%row(title,format)%]</b> - Заменяется на поле title в формате format (может быть nl2br, yes_no, if)<br>
<b>[%loop_begin%]</b> - Начало повторяющейся части кода<br>
<b>[%loop_end%]</b> - Конец повторяющейся части код<br>
<b>[%summ(title)%]</b> - Сумма значений поля title<br>

</font>
